MCP · destinos avioa (llama a las tools por el protocolo MCP en /mcp)


  

    

      tool
      get_destino
    

    
Devuelve el markdown del destino guardado en destinos.json (no re-scrapea). Código inválido → mensaje con sugerencias.

    

      
      
      
    

    

  


  

    

      tool
      buscar_vuelos
    

    
Cotiza vuelos en vivo en Travel Compositor (httpx, sin navegador): devuelve los más baratos (precio COP, horarios, aerolínea) y el enlace para ver todo y reservar. Retorno vacío = solo ida.
